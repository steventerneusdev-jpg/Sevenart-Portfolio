🎯 PROCHAINES ÉTAPES

1. REMPLACER LES VIDÉOS
   → Ouvrez src/App.jsx
   → Ligne 8 : tableau VIDEO_DATA
   → Remplacez par vos vraies vidéos YouTube

2. AJOUTER VOS MINIATURES
   → Mettez vos images dans src/assets/
   → Ou dans public/thumbnails/
   → Référencez-les dans VIDEO_DATA

3. TESTER
   → Ouvrez http://localhost:5174
   → Testez tous les filtres
   → Cliquez sur une vidéo pour tester la modal
   → Testez sur mobile (responsive)

4. DÉPLOYER
   → npm run build
   → vercel (ou netlify)
